# Insta Sales Demo

## 概要
このリポジトリは、営業時に飲食店向けSNS支援サービスを説明するための  
**Instagram風デモWebページ**です。

- 外部公開は想定していません
- 営業時に「完成イメージ」を伝えるためのデモ用途です

## このデモでできること
- 店舗情報（ジャンル・雰囲気・ターゲットなど）の入力
- 店舗画像の事前追加（外観 / 内装 / メニュー 等）
- AIによる9投稿分の画像・キャプション自動生成（デモ）
- Instagramと同じ見た目のUIで投稿プレビュー表示

## 注意事項
- 本リポジトリは **営業用デモ** です
- 本番用コードではありません
- デザインは説明用途のため、InstagramのUIに強く寄せています
- 外部への公開・配布は行いません

## 開発・修正について
- ローカル環境（Mac）での実行を前提としています
- 他メンバーが修正する可能性があります
- 不明点がある場合は、勝手に大きく構造を変えず共有してください
